CSS para Estructura de Sitio Web con Bootstrap

Archivo estilo.css

Este CSS debe guardarse en un archivo llamado estilo.css y enlazarse después de Bootstrap en el HTML.

Nota: Este CSS está diseñado para trabajar con la estructura HTML de Bootstrap y las clases que se muestran en el ejemplo.
/* estilo.css */ body { padding: 20px; font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; background-color: #f8f9fa; } .header { background: linear-gradient(135deg, #6a11cb 0%, #2575fc 100%); color: white; padding: 20px; border-radius: 8px; margin-bottom: 20px; text-align: center; } .sidebar { background-color: #343a40; color: white; padding: 20px; border-radius: 8px; height: 100%; } .main-content { padding: 0 15px; } .box { background-color: #74b9ff; color: white; padding: 20px; border-radius: 8px; margin-bottom: 15px; text-align: center; height: 120px; display: flex; align-items: center; justify-content: center; } .box-large { background-color: #17a2b8; height: 255px; } .box-sm { background-color: #74b9ff; height: 120px; margin-bottom: 15px; border-radius: 8px; display: flex; align-items: center; justify-content: center; color: white; } .box-sm-large { background-color: #17a2b8; height: 120px; } .d-sm-only { display: none; } @media (max-width: 767.98px) { .d-md-only { display: none; } .d-sm-only { display: block; } }

Explicación de las clases CSS

Clases principales:

  • .sidebar - Estilo para la barra lateral con fondo oscuro
  • .box - Estilo para los 6 rectángulos azules
  • .box-large - Estilo para el rectángulo celeste grande
  • .box-sm y .box-sm-large - Versiones para dispositivos pequeños

Clases de utilidad:

  • .d-md-only - Muestra elementos solo en resoluciones medianas y superiores
  • .d-sm-only - Muestra elementos solo en resoluciones pequeñas
  • .main-content - Espaciado para el contenido principal

Media Query:

El media query @media (max-width: 767.98px) controla el cambio entre los dos diseños:

  • Oculta el diseño para desktop (.d-md-only)
  • Muestra el diseño para móviles (.d-sm-only)

Instrucciones de uso

  1. Crear un archivo llamado estilo.css
  2. Copiar todo el contenido CSS del recuadro superior
  3. Pegarlo en el archivo estilo.css
  4. Enlazar el archivo CSS después de Bootstrap en el HTML:
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <link rel="stylesheet" href="estilo.css">
